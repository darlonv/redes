

# Roteamento

Entendemos como roteamento sendo a decisão de qual caminho deve ser percorrido por um pacote de sua origem até seu destino. Este é parte do *software* presente na camada de rede.

## Algoritmos de roteamento

DV - *Distance Vector*
    De acordo com o número de saltos

ADV - *Advanced Distance Vector*

LS - *Link State*
    Considera a velocidade entre os links
