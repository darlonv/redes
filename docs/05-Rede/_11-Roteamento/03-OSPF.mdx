# OSPF

O OSPF (*Open Shortest Path First*) é um algoritmo de roteamento, projetado para que seja possível determinar o menor caminho para o envio de pacotes em uma rede IP. 

O algoritmo OSPF é um algoritmo dinâmico, ou seja, os roteadores trocam informações entre si e com isso atualizam suas tabelas de repasse. Também e um algoritmo de roteamento interno, que pode ser utilizado em redes de médio e grande porte.
